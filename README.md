## 🛠️ Prerequisites

Before the workshop, please follow these steps to get ready:

### 1. Sign Up for a GitHub Account
If you don’t already have one, sign up here: [github.com/join](https://github.com/join)

---

### 2. Install Python (Windows & Mac)

AutoGen Studio requires Python (version **3.8–3.12**). _Installing version **3.12** would be ideal_.

#### Windows:
- Go to [python.org/downloads](https://www.python.org/downloads/)
- Download the latest version for Windows (make sure it’s 3.8, 3.9, 3.10, 3.11, or ideally **3.12**).
- Run the installer.
- **Important:** Check the box that says **“Add Python to PATH”** during installation.
- Complete the installation.

#### Mac:
- Go to [python.org/downloads](https://www.python.org/downloads/)
- Download the latest macOS installer (version make sure it’s 3.8, 3.9, 3.10, 3.11, or ideally **3.12**).
- Open the `.pkg` file and follow the prompts.
- By default, Python is added to your PATH. If you have issues, restart your Terminal or computer.

**To verify your installation, open Terminal (Mac) or Command Prompt (Windows) and type:**

Copy code

`python3 —version`

You should see a version number between 3.8 and 3.12.

> **Note for Mac users:** If `python` or `pip` doesn't work, then use `python3` and `pip3` in your commands (not `python` or `pip`).

> **Important:** AutoGen Studio v0.15 does **not** support Python 3.13 or newer. Use only versions 3.8–3.12.
> See [AutoGen Studio Getting Started](https://microsoft.github.io/autogen/0.2/docs/Getting-Started#installation).

---

### 3. **Install pip (Python package manager)**

Most Python installations include pip by default.

To check if you have pip, open your command prompt or terminal and type:

`pip3 —version`

If you see a version number, you’re good! If not, see [pip installation guide](https://pip.pypa.io/en/stable/installation/).

---

### 4. **Install AutoGen Studio**

Once Python and pip are installed, open your command prompt or terminal and run:

**Windows:**
`pip install autogenstudio==0.15`

_or_

**Mac**
`pip3 install autogenstudio==0.15`

For more details, see the [official Getting Started guide](https://microsoft.github.io/autogen/0.2/docs/Getting-Started).

### 🛠️ GitHub Copilot Installation Guide

#### Step 1: Activate Your Employee GitHub Copilot Benefit
- Go to [https://aka.ms/copilot](https://aka.ms/copilot)
- Sign in using the GitHub account you just created.
- Follow the instructions to activate your GitHub Copilot benefit as an employee.

#### Step 2: Sign In to GitHub Copilot in VS Code
- Open Visual Studio Code.
- Click the Extensions icon on the sidebar or press `Ctrl+Shift+X`.
- Search for “GitHub Copilot” and install it if needed.
- You should see GitHub Copilot on the right; click **Sign in to GitHub** when prompted.
- Use the GitHub account with the activated Copilot benefit to complete the sign-in process.

#### Step 3 (Optional): Download and Install Visual Studio Code (this step is not needed but I have a feeling you will get your fingers busy with coding after this!)
- Go to the official VS Code download page: [https://code.visualstudio.com/](https://code.visualstudio.com/)
- Choose your operating system (Windows, macOS, or Linux).
- Download and run the installer.
- Follow the prompts to complete the installation.

✅ **You’re All Set!**
You should now see GitHub Copilot suggestions as you type in VS Code. If you don’t, try restarting VS Code or checking the extension settings.
